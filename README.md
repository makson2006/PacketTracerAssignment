SafeHarbour Secure Network Infrastructure Redesign
Description
This project focuses on the end-to-end design and implementation of a modern, secure, and scalable network infrastructure for a mid-sized financial consulting firm. The project addresses critical vulnerabilities—including a previous ransomware incident—by replacing an outdated, flat Layer 2 network with a segmented, high-performance architecture.
